# Apartment Desktop Landing Page

A high-fidelity, production-ready landing page component inspired by Apartments.com.

## Features
- **Immersive Hero Search**: Large background visual with a complex search interface (Tabs + Input + Dropdowns).
- **Trust Indicators**: Value proposition section with icons.
- **Featured Carousel**: Grid of property cards showing "Trending" items.
- **Lifestyle Navigation**: Visual grid for browsing by category.
- **Responsive Design**: Fully responsive layout using Tailwind CSS.

## Usage

```tsx
import ApartmentLandingDesktop from '@/sd-components/c3ff673e-2fb9-4e77-b466-bc7531acf93d';

function HomePage() {
  return <ApartmentLandingDesktop />;
}
```

## Dependencies
- lucide-react
- framer-motion
- tailwindcss
