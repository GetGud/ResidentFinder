# Apartment Landlord Dashboard

A professional, high-density dashboard for property managers to track leads, occupancy, and portfolio performance.

## Features
- **Pro Dark Sidebar**: Distinct visual identity for the business side of the platform.
- **Analytics Overview**: Key performance indicators (Occupancy, Revenue, Leads) at a glance.
- **Lead Management**: Interactive table showing recent prospects and their status.
- **Property Quick View**: Summary cards for active properties in the portfolio.
- **Visual Charts**: CSS-only bar charts for traffic visualization.

## Usage

```tsx
import ApartmentLandlordDashboard from '@/sd-components/a65db75e-6531-4606-8340-a90f902778b7';

  return <ApartmentLandlordDashboard />;
```
