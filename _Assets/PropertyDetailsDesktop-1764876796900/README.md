# Property Details Desktop Component

A high-fidelity, conversion-focused real estate property details page.

## Features
- **Bento-Grid Gallery**: Interactive 5-image layout with hover effects.
- **Sticky Sidebar**: Lead capture form and tour scheduler that stays in view.
- **Rich Content Layout**: Clean typography with detailed amenities, floor plans, and description.
- **Micro-interactions**: Subtle hover states, saved toggle, and sticky header.

## Usage

```tsx
import { PropertyDetailsDesktop } from '@/sd-components/5e4c3370-541c-4392-b772-dab331e40b65';

function Page() {
  return <PropertyDetailsDesktop />;
}
```
